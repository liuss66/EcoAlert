## 本次功能

- 新增「域检测」：分组标题栏提供开关。
- 开启域检测后，分组内所有启用视频源均进入报警态时，才发送外部报警触发通知。
- 域检测保留单路报警状态展示，便于管理员定位是哪一路画面出现无人亮灯风险。
- 任一路从报警态恢复后，域检测分组发送一次恢复通知并重置整组通知状态。

## 运行前检查

- Windows 10 / 11。
- WebView2 可用。
- 如需检测、推流或 RTSP / HLS 抽帧，请确认 `ffmpeg -version` 可正常执行。
- 如使用仓库测试视频，请先启动 `Tools/push_streamer`。

## 推荐启动测试

```bash
cd Tools
pip install -r requirements.txt
python -m push_streamer.cli --config config.example.yaml
```

然后在 EcoAlert 中添加 HLS 视频源：

```text
http://127.0.0.1:8080/cam-1/index.m3u8
```

## 当前限制

- 人员检测当前仍为 ROI 内运动代理，不等同于生产级人形检测。
- 真实 VLM provider 尚待接入。
- RTSP 需要后续转码链路完善。
- 当前不包含录像存储、多用户权限和时间轴回放。
