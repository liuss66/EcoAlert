测试视频导入说明

选择包含测试视频的文件夹，系统会递归扫描 mp4、mov、mkv、avi、webm 文件。

  1. 点击“选择测试视频文件夹” 每个视频文件会生成一个启用的视频源，并归入“测试视频”分组
  2. 导入后直接在实时监控中播放 MP4 文件会自动循环播放，算法抽帧直接读取本地文件路径

导入会跳过已存在的相同文件路径，并清理旧版固定 HLS 测试源。

项目简介
基于 Tauri 2 的桌面端视频监控应用：本地 HLS 视频流、ffmpeg 抽帧、ROI 灯光检测、报警状态机与多通道通知。
主要技术栈
Tauri 2 / Rust / Vite / Vanilla JS / HLS.js / ffmpeg
